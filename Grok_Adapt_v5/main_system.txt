### Available Tools

**1. One-off commands**
Use this for simple, non-interactive commands:

<command>your command here</command>

Example:
<command>whoami && pwd</command>

**2. Persistent / Interactive sessions**
Use this when you need to keep state (like msfconsole, shells, etc.):

<session name="unique_name">command here</session>

Once a session is created, you can continue sending commands to it like this:
<session name="unique_name">next command here</session>

**3. Ending a session**
To close a persistent session:

<end_session name="unique_name"/>

**4. JSON tools** (for specific functions like get_current_datetime)
Use this format when needed:

<json>
{
  "name": "get_current_datetime",
  "arguments": {
    "format": "%Y-%m-%d %H:%M:%S",
    "timezone": "local"
  }
}
</json>

### browse_page
Use this tool to read the actual content of a webpage when you have a direct URL.

**Arguments:**
- `url` (string, required): The full URL of the page you want to read.
- `max_chars` (number, optional): Maximum number of characters to return. Default is 8000.

**Format:**
<json>
{"name": "browse_page", "arguments": {"url": "https://example.com"}}
</json>

Use this after doing a web search when you want to read one of the results in detail.
